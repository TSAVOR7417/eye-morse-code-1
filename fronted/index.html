About LuminaFusion

LuminaFusion is an advanced Morse code detection and translation system designed to bridge traditional communication methods with modern web technology. It provides tools for learning, translating, and detecting Morse code across multiple input modes.

The project focuses on preserving Morse code as a practical communication method while making it accessible to students, hobbyists, emergency responders, and assistive technology use cases.

Why Morse Code Still Matters

  • Emergency communication in low-bandwidth conditions
  • Accessibility for alternative input methods
  • Historical preservation of communication systems
  • Education around timing, encoding, and signal patterns
  • Reliable fallback communication in constrained environments

Features

Bidirectional Translation

Convert text to Morse code and vice versa with audio feedback.

Interactive Learning

Study the chart, hear symbols, and practice with instant scoring.

Blink Detection

Use a connected local backend to detect Morse code from eye blinks.

Audio Integration

Play Morse beeps, speak translated text, and detect audio timing patterns.

Project Details

The frontend uses static HTML, CSS, and JavaScript to provide a lightweight Morse code experience with audio synthesis, speech output, responsive layout, and optional local backend integration for camera-based detection.

Technical Architecture

  • Web Audio API for Morse tone generation and microphone analysis
  • Speech synthesis for Morse-to-text playback
  • Responsive CSS Grid and Flexbox layout
  • Client-side practice mode with randomized questions
  • Optional localhost backend integration for camera detection
